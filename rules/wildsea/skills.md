# Skills

- Each skill has a **rank** from 1-3; it adds that many **d6s** to your dice pool when used in a roll
- Skills are flexible by design
- Not having a skill doesn't block you from trying - skills make you **better**, not permitted
- Two crewmembers can work together on the same task. Both are affected by any negatives from a **conflict** or **disaster**.
