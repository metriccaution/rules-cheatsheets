# Languages

Like skills, languages have ranks from 1-3. Unlike skills, ranks are named:

| Rank | Name       | Ability                                                                   |
| ---- | ---------- | ------------------------------------------------------------------------- |
| 1    | Smattering | Simple words and sentences; common topics, names                          |
| 2    | Knowledge  | Normal conversation; wide range of topics, opinions, directions           |
| 3    | Fluency    | Complete mastery; read, write, listen, and speak without misunderstanding |

- You rarely need to roll - languages are used to **draw on cultural knowledge** and **gain information**
- **Dialects**: the one case where the Firefly may call for a roll - encountering a dialect far removed from the language you know; even fluent speakers can be confused
